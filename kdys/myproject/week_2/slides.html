What makes a computer function ?

Hardware

The HARD parts, these are the parts you can see and touch.


Software

The bits and bytes, the computer code, Software cannot be touched or seen !

Again, this concept can be understood by comparing a computer to our own bodies and our senses. If our nose is the hardware, our sense of smell is the software, which is connected to the brain (CPU)! Similarly, our eyes are hardware, while sight is software.

Software

A computer can’t think for itself, in order for it to do anything it needs to be told what to do and how to do it.
It has been “programmed” by a human, these programmes or instructions are known software.

Two main types of software

  • Operating System
    • The software utilities which you need to get your computer going and manage your files, and takes input from your mouse and keyboard.
  • Appliaction programmes
    • These are the software you use to do stuff, for example a browser to surf the web, a word processor to write a letter, or a movie player to watch a video.

What is an operating system?

  • Without one, all you've got is the hardware & BIOS - not very useful!
  • An operating system manages your computer for you.
  • An operating system allows higher-level applications to run on your computern
  • It does jobs like 'saving files' for applications

Common operating system

  • Microsoft - Windows
  • Apple - Mac OS
  • Linux

What's Free and Open Source got to do with it?

  • Free Software is free.
  • Free Software is user friendly.
  • Free Software is a community.
  • Free Software provides an operating system and it provides applications.
  • Free Software lets you be more productive at less cost.
  • Free Software lets you share it with your friends.
