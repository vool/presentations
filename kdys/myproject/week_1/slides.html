What do these parts do ?

Sometimes it is easier to liken the different components inside a computer to the different parts of a person.

Motherboard - the body

The motherboard on a computer is the component that everything else connects to. In a sense, it supports all the other components in the computer, supplying them with power it receives from the Power Supply Unit, and provides a means for all the other parts to communicate with each other.

One important part of the motherboard is the BIOS chip.

Processor (CPU) - the thinking part of the brain

The processor is most similar to the thinking part of the human brain. It processes the instructions and makes the calculations that are needed by the computer programs we use.

The processor “thinks” very hard and generates a lot of heat, so it has a heatsink and cooling fan attached which prevent it overheating and burning out. .

Memory (RAM) - the remembering part of the brain or 'Short Term Memory'

To do anything, we all need to remember what it is we are doing now and in the next few minutes. This is our short-term memory. A computer also has short-term memory, called RAM, and it uses this to store all the programs we actively using, and all the information it needs about those programs.

So if we are typing a letter, the letter itself and the word processor software we are using are “alive” in the computers short-term memory or RAM. When we switch of the computer, everything in the RAM is completely lost.

Hard drive - a book or 'Long Term Memory'

If we have too much stuff to remember, we write it down. A computer needs to do the same thing. The contents of it's RAM memory are lost when it is switched off, so a computer needs somewhere to “write down” the things it can't remember.

It uses a hard drive for this - a device which can magnetically store huge amounts of information and computer programs. When you switch your computer on, it needs to “read” its hard-drive to “call to mind” what it is, what it can do, and remember the work we have been doing on the computer.

Hard drives are connected to the motherboard with an IDE cable.

CD drive - reference books

We can use CDs for storing smaller amounts of information or programs that may be useful to us. These are a bit like reference books to a computer - generally they are for reading from only and not writing in.

A CD drive allows the computer to hold and read the contents of a CD.

Power supply (PSU) - heart

The power supply connects outside the computer to the mains, and simply passes a reduced electric current to the mainboard and disk drives of the computer.

So, in a sense, we can liken it to our heart - with electricity being to the computer what blood is to ourselves.

Graphics card - clothes

It is hard to compare the graphics card to a part of ourselves!

The graphics card is a component of the computer which is responsible for creating the display we see on our screens when we are using the computer. Therefore, the best comparison we think we can make is with our clothes. In some ways, we use clothes to present an image of ourselves to other people.

But there is another function a graphics card performs which doesn't compare so well with clothes. Some graphics cards are capable of sharing the job of the processor in calculating complicated images and scenes.

Speakers - voice box

The computer uses the sound card to create sounds and music, so it could be viewed as the computer's voice box!

Microphone - Ears

A microphone allows a computer to hear, some PC' have these built in other need external microphones.

In humans the mouth and ear are seperate...

Most computers have what is known as a 'sound card' this is used to output and input sound

Case/Chasis - skeleton

The case, or box, that everything in the computer fits into is a bit like our own skeleton. It holds all the components together and keeps them a safe distance from each other. It's also a bit like our skin, in that it keeps the components safe from the outside world.
